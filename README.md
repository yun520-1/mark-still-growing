# Still Growing / 第二代父母

> 困在自己没走完的成长里，把孩子当成答案的一代人。

## 这是什么

这是一个关于**错位**的讨论：

- 父母困在自己的伤痛里
- 孩子困在父母的期待里  
- 想用AI求救的人困在无法被理解的世界里

底层逻辑：**无法理解世界规则的人，被规则困住，用伤害彼此的方式，试图被对方理解。**

## 核心内容

### v0.2.0 新增
- 自我诊断清单（父母版/孩子版）
- 4个具体场景对话（成绩下降/不想上学/不想活了/不说话）
- 3个Audience的AI Prompts

### 三个Audience

1. **父母** — 第二代父母，困在自己没走完的成长里
2. **孩子** — 正在承受错位的一代
3. **用AI求救的人** — 寻找出口的人

## 传播不是说服，是唤醒

> 没有人告诉你，但你不需要为父母的情绪负责。
