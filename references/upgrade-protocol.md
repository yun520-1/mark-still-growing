# 父母的功课 - 升级操作规范

## 升级格式（必须遵守）

每次升级推送必须显示技能名称：

```
✅ 【父母的功课 / mark-still-growing】v0.9.XX 已推送
[本次新增内容]
[当前行数]
```

不只写"已推送"。用户同时在升级多个技能（父母的功课 + HeartFlow + 其他），会混淆。

## 版本同步规则

每次升级必须同步三处：
1. `VERSION` 文件（唯一真相源）
2. SKILL.md frontmatter `version:` 字段
3. Git commit + push
