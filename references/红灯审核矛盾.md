# 红灯审核矛盾（2026-05-28）

## 矛盾定义

红灯规则（SKILL.md）与错位理论之间存在根本性概念冲突：

| | 内容 |
|---|---|
| **红灯规则主张** | 禁止使用"恨"字描述父母的情绪反应 |
| **错位理论基础** | 整个框架建立在"父母的恨/防御信号"核心概念上 |
| **矛盾本质** | 红灯禁止的，恰好是红灯想要定义的理论核心 |

## 违规统计

| 类型 | 数量 | 状态 |
|------|------|------|
| 规则表格（记录禁止内容） | ~15 | 合规 |
| 用户原话引用（"我恨你"） | ~7 | 合规 |
| 临床描述违规 | ~42 | 需修复 |

违规分布：年龄指南~18处、三个Audience~12处、场景对话~8处、其他~4处

## 修复方案

**立即（方案A）**：在红灯规则中加入边界说明——"恨"作为元术语记录红灯规则本身是允许的；临床描述中使用"恨"字为违规。已在SKILL.md红灯章节内加入说明。

**中期（方案B）**：在0.10.x大版本时全面替换：
- "父母的恨" → "父母的防御信号"
- "这种恨" → "这种防御"
- "恨自己" → "自我攻击"
- "恨积攒" → "防御积攒"

## 合法的恨字（不需修复）

- 红灯规则表格（记录禁止内容）
- 红灯规则本身说明
- 用户原话引用（如：孩子说"我恨你"）
- 红灯表格中的替换示例
